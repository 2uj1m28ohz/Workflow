# :infinity: Perguntas Frequentes

1. O software coleta dados?

    O software coleta e armazena localmente informações sobre o ambiente e o usuário durante a execução, no Registro de Eventos. Esses dados são automaticamente excluídos conforme a Política de Retenção. O usuário pode desativar o Registro de Eventos ou ajustar o período de retenção nas configurações.

2. Como os dados do Registro de Eventos são usados?

    Os dados do Registro de Eventos ajudam a monitorar o desempenho, diagnosticar problemas e otimizar o software. Eles permitem identificar padrões de uso, aprimorar a experiência do usuário e manter um registro histórico das atividades do software, facilitando a análise retroativa e a recuperação de informações.

3. O software envia dados pela internet?

    Nenhum dado é enviado para a internet.

4. O software será compatível com outras plataformas?

    O suporte ao Linux está em desenvolvimento, com versões experimentais sendo lançadas para avaliar a compatibilidade coletar feedback. Essas versões podem apresentar instabilidades e limitações.

    > A disponibilidade de recursos e funcionalidades pode variar conforme a plataforma.

5. O software terá suporte a outros idiomas?

    Não há planos para suporte a outros idiomas no momento.
